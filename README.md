# Expense Tracker (CLI)

A simple CLI-based expense tracker built using Python and OOP concepts.

## Features
- Add transactions (income/expense)
- View all transactions
- Search transactions by category
- Delete transactions
- Calculate net balance
- Data stored using JSON

## Tech Used
- Python
- OOP (Classes & Objects)
- File Handling (JSON)

## How to Run

python main.py
